Wlan Detektor Pro

Wlan Detektor Pro ist eine Applikation bei welcher die Wlan Signalstärke gemessen werden kann. Sie wurde im Zuge einer Diplomarbeit eines Nachdiplomstudiums an der ABB Technikerschule entwickelt.
